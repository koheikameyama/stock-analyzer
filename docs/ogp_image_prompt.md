# OGP画像作成プロンプト（Canva用）

## 画像サイズ
1200 x 630 px

## デザイン指示

### 背景
- ベースカラー：濃紺（Navy Blue #1e3a8a）からエメラルドグリーン（#10b981）のグラデーション
- 左上から右下に向かって滑らかなグラデーション

### メインテキスト（中央上部）
- 「AI株式分析ツール」
- フォント：太字、ゴシック体
- サイズ：大きく目立つように
- カラー：白（#FFFFFF）

### サブテキスト（中央）
- 「AIによる株式投資判断サポート」
- または「データドリブンな投資判断を、あなたに」
- フォント：中太、ゴシック体
- サイズ：メインテキストの60%程度
- カラー：白（#FFFFFF）または薄い緑（#d1fae5）

### 特徴リスト（中央下部）
- 「✓ 毎日自動更新」
- 「✓ 詳細な分析根拠」
- 「✓ 投資判断をサポート」
- チェックマーク（✓）のみ：エメラルドグリーン（#10b981）
- テキスト部分：白（#FFFFFF）
- フォント：通常の太さ、ゴシック体
- 左寄せまたは中央寄せ

### URL（最下部）
- 「stock-analyzer.jp」
- フォント：中太
- サイズ：やや小さめ
- カラー：白（#FFFFFF）または薄い緑（#d1fae5）

### ビジュアル要素（右側または背景）
- 上昇トレンドのチャート/グラフのシルエット
- AIを象徴する回路パターンや光のエフェクト
- 不透明度：20-30%（背景に溶け込むように）
- カラー：白または薄い緑

### 左上にロゴ配置（オプション）
- サイズ：80x80px程度
- 位置：左上コーナーから少し余白を取る

## デザインのポイント
- シンプルで読みやすく
- テキストは中央寄せで配置
- 余白を十分に取る
- プロフェッショナルな印象
