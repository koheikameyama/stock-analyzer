## 1. 中核ビジョン

### 1.1 ミッション

> **「誰もが、自分の未来に自信を持って投資できる世界を作る」**

### 1.2 現状認識

**投資アドバイスのアクセス格差**

```
富裕層（資産1億円以上）
  └─ プライベートバンク、専属アドバイザー
     月額数十万円のコンサルティング

中間層（資産1,000万円〜）
  └─ 証券会社の対面相談
     営業色が強い、万人向けの推奨

一般層（〜1,000万円）
  └─ ネット証券のロボアドバイザー
     機械的、個別対応なし

初心者
  └─ 情報収集のみ（Yahoo!ファイナンスなど）
     分析・判断は自力
```

### 1.3 解決したい課題

#### 課題1: 情報の非対称性
- 専門家と初心者の知識格差が大きすぎる
- 財務諸表を読めない人が大多数
- 「どの銘柄を買えばいいかわからない」

#### 課題2: 相談相手の不在
- 証券会社: 営業目的で中立性に疑問
- ロボアド: 機械的で「なぜ？」に答えてくれない
- 友人・家族: 専門知識がない

#### 課題3: 学習曲線の高さ
- 投資を学ぶのに時間がかかる
- 専門用語が多く挫折しやすい
- 実践しながら学ぶ環境がない

#### 課題4: アクションの障壁
- 情報はあっても「次に何をすれば？」
- ポートフォリオ管理が煩雑
- 税金最適化（NISA活用）が複雑

### 1.4 提供する価値

#### 初心者向け
- ✅ わかりやすい説明（ツールチップ、平易な言葉）
- ✅ AI分析結果の可視化
- ✅ 「なぜこの推奨か」の理由提示
- ✅ いつでも質問できるAIアドバイザー
- ✅ 段階的に学べる教育コンテンツ

#### 中級者向け
- ✅ 多角的な分析（財務+ニュース+テクニカル）
- ✅ ポートフォリオ最適化提案
- ✅ カスタムスクリーニング
- ✅ リスク許容度に応じた個別推奨

#### 上級者向け
- ✅ 高度な分析ツール
- ✅ データAPI提供
- ✅ バックテスト機能
- ✅ アラート・通知機能

```
  └─ 日本語BERTでセンチメント分析

テクニカルデータ（20%）
  ├─ RSI（相対力指数）
  ├─ MACD（移動平均収束拡散）
  └─ ボリンジャーバンド

ニュースインパクト（10%）
  └─ 重要度加重
```

**Level 3: 高度なインサイト**（フェーズ3-4）
```
業界分析
  ├─ セクター全体のトレンド
  ├─ 業界内ポジション
  └─ 競合との相対評価

マクロ経済との相関
  ├─ 金利との相関
  ├─ 為替の影響
  └─ 景気サイクル

隠れた優良銘柄発掘
  ├─ AIが市場の非効率性を発見
  ├─ アナリストカバレッジが少ない銘柄
  └─ 成長性が過小評価されている企業
```

**Level 4: 予測・先読み**（フェーズ5、将来）
```
株価予測
  ├─ 機械学習モデル（LSTM、Transformer）
  ├─ 過去データからパターン学習
  └─ 確率的予測（上昇確率70%など）

イベント影響予測
  ├─ 決算発表の影響度予測
  ├─ 新製品発表の市場反応予測
  └─ M&Aの影響分析

市場の転換点検知
  ├─ トレンド転換の早期発見
  ├─ 過熱・冷え込みシグナル
  └─ セクターローテーション予測
```

#### データ → インサイト変換例

**従来の情報サイト**:
```
トヨタ自動車（7203）
PER: 10.5倍
PBR: 1.2倍
ROE: 11.2%
配当利回り: 2.8%

→ ユーザー: 「で、これは良いの？悪いの？」
```

**このサービス**:
```
トヨタ自動車（7203）- 買い推奨 ✅

【総合スコア】75/100（信頼度: 82%）
  財務スコア: 80/100（割安、ROE良好）
  センチメント: +45（ニュースは概ねポジティブ）
  テクニカル: 70/100（上昇トレンド）

【AIの判断理由】
自動車業界は電動化で転換期ですが、トヨタは
ハイブリッド技術で先行しており、2025年の
EV新モデル投入計画が市場で好感されています。
PER 10.5倍は業界平均12倍より割安で、
配当利回り2.8%も魅力的です。

【リスク】
- 為替変動（円高で減益の可能性）
- EV移行の遅れリスク

【類似銘柄】ホンダ（7267）、日産（7201）
```

→ ユーザー: 「なぜ買いなのか理解できた！」
